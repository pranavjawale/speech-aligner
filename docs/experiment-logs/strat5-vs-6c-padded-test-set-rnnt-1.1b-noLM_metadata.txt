PURPOSE
-------
NON-CIRCULAR companion to strat5-vs-6c-padded-test-set-ctc-0.6b-noLM. Same two
PADDED test splits, same decoding config, but eval with an INDEPENDENT model
(parakeet-rnnt-1.1b: RNN-T, no LM, NOT used in any alignment arm). This is the
HONEST label-quality comparison: a lower WER for an arm means its located reference
agrees more with a model that never saw the 6-gram / never built the labels.

labels -> home-field advantage). rnnt-1.1b removes that: it played no role in 6c.
CAVEAT: rnnt-1.1b is still Parakeet-family (shared FastConformer lineage with the
ctc-0.6b used in 6b/6c) — independent of the LM + window choice, but not a totally
different acoustic lineage. (Whisper is disqualified — it is used in every arm.)
