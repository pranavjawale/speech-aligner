PURPOSE
-------
Compare WER on the PADDED speaker-segment TEST split between the two label sets
(strategy-5 = WhisperX-only alignment; 6c = LM-assisted, now canonical) using ONE
fixed model + decoding config. Both test splits cover the SAME 20 test speakers
(6c reused strategy-5's speaker_assignment.json); the segments/audio/labels differ.

CIRCULARITY CAVEAT (read before interpreting): the eval model below
(parakeet-ctc-0.6b) is the SAME acoustic model that helped CREATE the 6c labels
(best-of-two; Parakeet+KenLM won ~70% of chunks). So a lower 6c WER here is partly
a home-field advantage, NOT pure label-quality proof. For a NON-circular
label-quality claim use an independent judge (parakeet-rnnt-1.1b) -> see
code/data-for-asr-baseline/results/noncircular_wer_6c_vs_strat5.txt and
code/phase13_indep_wer.py.

  - Overall ~TIED on corpus WER (-0.13pp); 6c modestly better on mean-utt (-1.14pp,
    i.e. 6c helps shorter utterances). 13 of 20 speakers improve, mostly small;
    only spk_1042 regresses notably (+4.4pp); no catastrophic per-speaker regressions.
  - The 4 bad-audio speakers (1019/1026/1031/1032) stay ~flat at 79-94% WER -> their
    labels are unreliable regardless of strategy (drop/quarantine before Phase 15).
  - CIRCULARITY: this is the CIRCULAR eval (ctc-0.6b helped build the 6c labels), so
    even the small 6c edge is partly home-field. The non-circular read
    (noncircular_wer_6c_vs_strat5.txt, rnnt-1.1b judge) is the honest label-quality
    comparison. Net: on the Phase-15-relevant padded-test WER, 6c ~= strat-5 overall.
